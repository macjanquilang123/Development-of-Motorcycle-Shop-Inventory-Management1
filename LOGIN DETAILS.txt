To run this project, you must have installed a virtual server i.e XAMPP on your PC (for Windows). 
After Starting Apache and MySQL in XAMPP, follow the following steps. 
1st Step: Extract file 
2nd Step: Copy the main project folder 
3rd Step: Paste in xampp/htdocs/ 
4th Step: Open a browser and go to URL “http://localhost/phpmyadmin/” 
5th Step: Then, click on the databases tab 
6th Step: Create a database naming “motorcycle_parts_db” and then click on the import tab 
7th Step: Click on browse file and select “inventorymanagement.sql” file 
	which is inside the “Development of Motorcycle Shop Inventory Management” folder 
8th Step: Click on go. After Creating Database, 
9th Step: Open a File with VScode and go to terminal.

terminal: cd backend
	  node server.js

	  cd frontend
	  npm run dev
